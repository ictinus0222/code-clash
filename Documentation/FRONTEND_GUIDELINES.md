# Design Style & Experience Guidelines

## 1. Design Style / Aesthetic

The CodeClash website follows a **modern experimental tech‑conference aesthetic** that combines:

- Minimalism
- Brutalism
- Futuristic tech design trends

The interface should feel **bold, energetic, and creative** rather than corporate.

Key visual characteristics:

- High contrast visuals
- Bold expressive typography
- Large hero sections
- Dynamic layouts
- Interactive visual elements

The goal is to create a **tech festival atmosphere** that excites developers and designers.

---

# 2. Layout & Structure

The website uses a **single‑page scrolling layout**.

### Structure

```text
Hero Section
     ↓
Rounds Section
     ↓
Rules Section
     ↓
Footer
```

### Layout Principles

- Large immersive sections
- Clear visual hierarchy
- Generous spacing
- Grid‑based layouts
- Asymmetric compositions
- Layered visuals

The layout should feel **editorial and storytelling‑driven**, guiding users through the event naturally.

---

# 3. Color Palette

The design relies on **high contrast color combinations**.

### Primary Background

Dark backgrounds create a futuristic tech environment.

Examples:

- Deep Black
- Midnight Blue
- Dark Purple

### Accent Colors

Bright neon accents create visual focus.

Examples:

- Neon Pink
- Electric Purple
- Bright Cyan
- Neon Blue


### Usage Rules

Accent colors should highlight:

- Buttons
- Key headings
- Call‑to‑action elements
- Important information

This palette creates a **tech‑festival vibe**.

---

# 4. Typography

Typography plays a major role in the design.

### Font Style

Bold modern **sans‑serif fonts**.

Recommended fonts:

- Inter
- Space Grotesk
- Satoshi
- Orbitron (for display)

### Typography Characteristics

- Oversized hero headings
- Strong weight contrast
- Tight spacing for headings
- Clean readable body text

### Hierarchy

Hero Title → Very large  
Section Titles → Large  
Body Text → Medium  
Supporting Text → Small

The result should feel like a **futuristic digital magazine layout**.

---

# 5. Buttons & UI Components

Buttons should be **high‑impact call‑to‑action elements**.

### Shape

Rounded or pill-shaped.

Example:
border-radius: 999px

### Button Style

- Bright accent background
- High contrast text
- Subtle glow effects

### Hover Effects

Possible interactions:

- Slight scale animation
- Glow highlight
- Color transition

Buttons must feel **interactive and playful**.

---

# 6. Interaction & Micro‑Animations

Motion design should make the website feel **alive**.

### Interaction Types

- Smooth scrolling
- Hover effects
- Scroll reveal animations
- Section transitions
- Animated hero elements

### Animation Principles

Animations should:

- Be smooth and fluid
- Guide user attention
- Enhance immersion
- Never distract from content

---

# 7. Overall User Experience

The user experience should focus on **visual storytelling**.

Instead of complex navigation menus, users explore the page through **engaging sections**.

### UX Goals

The experience should feel:

- Creative
- Immersive
- Modern
- Futuristic
- Interactive

The website should give the feeling of attending a **tech innovation festival** rather than visiting a typical event page.
